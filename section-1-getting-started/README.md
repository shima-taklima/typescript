# Getting Started

## 01 - What Is TypeScript?

TypeScript is a programming language created by Microsoft.

It is basically JavaScript + extra features that help you write safer and cleaner code.

## 02 - Why Would You Use TypeScript?

1. Catch Errors Early (Before Running Code)

2. Clear & Self-Documenting Code

3. Easier to Maintain Large Projects

4. Support Modern JavaScript Features

5. Works Perfectly with React & Angular
